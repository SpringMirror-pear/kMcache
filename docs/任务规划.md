## 7. 文档更新规则

后续每次完成任务时，必须同步更新以下内容：

1. 在“已完成任务清单”中新增或更新对应任务状态。
2. 在“未完成任务清单”中移除已完成任务，或把状态改为部分完成。
3. 如果新增测试，需要更新“当前测试状态”。
4. 如果实现路径或优先级变化，需要更新“建议的下一阶段执行顺序”。

建议状态值统一使用：

- `未完成`
- `部分完成`
- `已完成`
- `已通过`
- `未测试`
- `不适用`
